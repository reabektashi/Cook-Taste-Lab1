# Cook&Taste (Frontend + Backend)

This project has two parts:
- **Frontend** (React/Vite)
- **Backend** (Express + MySQL) running on **HTTPS**

## Prerequisites
- Node.js (LTS recommended)
- MySQL running locally or a remote MySQL DB
- (If using HTTPS locally) cert files in backend: `backend/certs/`

---

## Run Backend

```bash
cd backend
npm run dev
## Run Frontend
cd frontend
npm run dev
